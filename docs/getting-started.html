Getting Started

This is the fastest way to make a sound in SoundShop2.

1. Your first project

When SoundShop2 opens, you'll see a node graph with a few nodes already placed: a Computer Keyboard input, a MIDI Track, and a Master Out. On a new project, a wizard may also pop up asking which hardware MIDI devices to add — you can skip it for now.

Everything in SoundShop2 is a node. Cables between nodes carry data. What's already wired for you: the Computer Keyboard's MIDI output connects to the MIDI Track's input, so anything you type on your keyboard will land in that track.

2. Add an instrument

Right-click an empty area of the graph → Instruments → Built-in Synth → Waveform. A new synth node appears, and its waveform editor opens automatically. Close it for now (you can reopen it by double-clicking the node).

3. Wire it up

Drag from the MIDI Track's MIDI output pin (right side of the node) to the new synth's MIDI In pin. Then drag from the synth's Audio output to the Master Out's In pin.

The signal flow is now: Computer Keyboard → MIDI Track → Synth → Master Out. Each cable is colored by the type of data it carries (green for MIDI, blue for audio).

4. Play

Click anywhere in the graph area to make sure it has keyboard focus, then press keys on the home row. The keys A through L are white keys; W through P are black keys; Z and X shift the octave down and up.

If you have a hardware MIDI keyboard connected and added via the wizard, wire its node to the MIDI Track the same way, and playing it works too.

Tip: Shift+key plays louder, Alt+key plays softer. That gives typed notes some dynamic range.

What next?

  • Read Graph Basics to understand the different kinds of cables
  • Open the synth's waveform editor (double-click) to shape the sound — see Wavetables
  • Double-click the MIDI Track to open its piano roll and write a melody instead of typing it live
  • Right-click an empty area and explore the menus — add effects, signal sources, or more tracks
