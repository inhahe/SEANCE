Layers & Effect Groups

Home Graph Basics

Most DAWs treat "which effects are active" as a fixed property of a track. In SoundShop2 you can make that time-varying — a given effect wire can be on only during certain beat ranges, and off otherwise. This section is about how that works.

The problem

Suppose you have a synth track feeding into a reverb, and you want the reverb active only during the chorus. In a traditional DAW you'd automate a dry/wet knob. In SoundShop2 you can instead mark the wire between the synth and the reverb as time-gated, with bright-colored regions showing where the gate is open.

Layers (effect regions)

A layer is a time range during which a specific cable is active. You can see layers as colored bars drawn above the notes in the piano roll, each bar representing one layer. Outside the bar, the wire is muted (with a short crossfade at the edges to prevent clicks — see Options → Crossfade Duration).

To create a layer:

  1. Right-click on a wire in the graph, choose Time-gate…
  2. Drag in a piano roll layer bar area to set the start/end of the region
  3. The region is tied to this specific wire, so it only affects that one connection

Each layer has a color matching the wire (or group) it belongs to so you can tell at a glance which routing is being gated.

Effect groups

Sometimes you want to activate multiple wires at once — e.g., a reverb send AND a delay send AND a filter mod, all gated as one "chorus effects on" layer. That's what an Effect Group is: a named bag of wires that can be activated together.

To create an effect group:

  1. Right-click a wire → Effect Group → New Group…
  2. Give it a name
  3. Right-click other wires → Effect Group → Add to [group name]

Now you can create a layer that references the group instead of a single wire, and all wires in the group activate/deactivate together. Each group gets its own color drawn on member wires (circles for wire identity, diamonds for group membership).

The routing strip

Above the piano roll, a narrow routing strip appears when any layers exist. It shows each gated wire as a horizontal "wire" bar with 3D shading, colored to match its wire/group. The horizontal axis matches the piano roll below, so you can see at a glance which wires are active at which beats across the whole project.

Crossfade edges

When a layer turns on or off, the wire crossfades over a short time so you don't hear a click. The default is 50 ms, set globally via Options → Crossfade Duration. An effect group can override the global duration if it has a non-zero crossfadeSec value, otherwise it inherits.

Tip: layers are a good way to think about arrangement differently than "track A is always there, track B comes in later." Instead of duplicating tracks, route everything through one track and use layers to gate routing on and off over time.
