SEANCE is a node-based digital audio workstation designed for people who don't have a traditional music background. Instead of hiding signal flow behind menus, every building block is a node on a graph, and you wire them together with cables to route MIDI, audio, and control signals.

SEANCE is in active development. These docs cover the major nodes and concepts, but new features sometimes ship before their docs are written — if a node has no docs page yet, hover its controls for tooltips and check Help → Keyboard Shortcuts.
