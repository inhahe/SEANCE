Piano Roll

The piano roll is where you see and edit the notes inside a MIDI Timeline clip. Open it by double-clicking a MIDI Track node — the editor opens as a panel at the bottom of the window. Each note is a horizontal bar at the row of its pitch, starting at the bar's left edge and lasting until its right edge.

Note editing basics

  • Click empty space to place a new note at that pitch and beat. The note takes the most-recent length you set.
  • Drag a note to move it in time and pitch.
  • Drag the right edge of a note to resize it. Drag the left edge too if you want to change where it starts.
  • Click+drag in empty space to box-select multiple notes at once.
  • Right-click a note for the per-note menu (delete, copy, paste, change pitch, etc.).
  • Right-click empty space for the canvas menu.
  • Delete key removes whatever's selected.
  • Hold Alt while dragging to bypass snapping for fine placement.

The piano keyboard down the left edge shows you the pitch of each row. Click a key to audition the synth at that pitch — useful for hearing how a synth sounds without typing on the computer keyboard or hooking up a MIDI controller.

Snap

The snap row in the toolbar (1/4, 1/2, 1, Off) controls how new notes and dragged notes line up to beats. 1 snaps to whole beats, 1/2 to half-beats, 1/4 to quarter-beats. Hold Alt while dragging to temporarily disable snapping no matter which mode is active.

The Snap to Scale button takes the snap idea one step further: when on, dragging a note up or down only lets it land on pitches that fit the chosen Key/Scale. Useful for "in-key" composition without having to think about which black keys to avoid.

Key, Scale, Mode, Root

The four dropdowns in the toolbar set the musical key context for the clip. They affect which notes get highlighted as "in scale" on the piano roll background, and they're what Snap-to-Scale uses.

  • Root — the home pitch (C, C#, D, …). For C Major it's C; for A Minor it's A.
  • Key — the family (Major, Minor, etc.). Major sounds happy/bright, Minor sounds sad/dark.
  • Mode — variants of the major scale that change the mood. Ionian is the standard major; Dorian, Phrygian, Lydian, Mixolydian, Aeolian (= natural minor), and Locrian each shift the pattern of whole and half steps for a different flavor.
  • Scale — broader categories: Pentatonic (5 notes, hard to sound wrong), Blues, Whole-Tone, Chromatic (all 12 notes — turns off scale highlighting), and so on.

If you don't already know what scale your existing notes are in, click Detect Key and the editor will analyze the notes and guess the most likely key/scale, then set the dropdowns to match.

Toolbar transformations

Select some notes (or none — the buttons act on the whole clip if nothing is selected), then use the toolbar buttons to transform them:

  • +Octave / -Octave — moves selected notes up or down by 12 semitones
  • +Semitone / -Semitone — moves selected notes up or down by 1 semitone
  • Nudge Left / Nudge Right — moves selected notes earlier or later by one snap unit
  • x2 Duration / /2 Duration — doubles or halves the length of selected notes
  • Reverse — flips the time order of selected notes (last becomes first)
  • Detune slider — applies a fine pitch offset in cents (1/100 of a semitone) to selected notes. Use for subtle out-of-tune effects or to match another instrument's tuning.

Mute, Solo, Pan

The Mute and Solo buttons in the toolbar work on this MIDI Timeline node. Mute silences it; Solo silences every other non-soloed track. The Pan slider sends this track's audio left or right in the stereo field — though for a MIDI Timeline that controls a synth, it's the synth's audio output that gets panned, not the MIDI events themselves.

Lanes (the panel below the notes)

Below the note grid you can show one extra lane of data per note. Pick which lane is visible from the lane buttons:

  • Off — hides the lane to maximize note grid space.
  • Vel (Velocity) — one vertical bar per note, spanning the note's duration. Bar height shows velocity (0-127). Drag a bar up or down to change that note's velocity.
  • Automate Param dropdown — pick any parameter on the parent node, and a continuous automation curve appears in the lane. Click in the lane to add control points; drag points to shape the curve. The curve uses Catmull-Rom smoothing between points. The synth reads the parameter from the curve during playback whenever this clip is playing.

Mouse and keyboard navigation

  • Mouse wheel — scroll the pitch axis up/down
  • Shift + wheel — scroll the time axis left/right
  • Ctrl + wheel — zoom horizontally toward the cursor
  • Horizontal zoom slider — at the bottom; gives you a precise zoom value if scroll-wheel zoom doesn't get you there

Compact mode

Click the -- button at the top-left to toggle compact mode, which hides most of the toolbar to give you more vertical space for the note grid. Click ++ in compact mode to bring the full toolbar back.

Undo

Note edits (placement, deletion, drag, transform) all create undo steps automatically. Ctrl+Z to undo, Ctrl+Shift+Z (or Ctrl+Y) to redo. Undo history persists across project save/quit, so you can keep undoing past your last save in the next session.
