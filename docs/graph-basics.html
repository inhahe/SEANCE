SEANCE's main view is a node graph. Each box is a node — a thing that does something. Cables between nodes carry data from one to another. The direction of data flow is always from a node's output pin (right side) to another node's input pin (left side).

Nodes

Nodes come in a few different categories, color-coded:

  • Input — a source of data. Computer Keyboard, MIDI devices, audio inputs, XY pads. Has only output pins.
  • Timeline — stores notes or audio clips over time. MIDI Timeline, Audio Timeline. Playback is driven by the transport.
  • Instrument — turns MIDI into audio. Built-in synths, sampled instruments (SF2/SFZ), drum synths.
  • Effect — transforms audio or MIDI as it passes through. Reverb, filter, compressor, trigger, MIDI modulator, etc.
  • Signal shape — generates control signals (LFOs, envelopes, XY pads). These modulate params on other nodes.
  • Output — where audio leaves SEANCE (speakers, Master Out).

Pins and cables

Each node has pins on its left (inputs) and right (outputs). Pins are color-coded by the kind of data they carry, and cables match the pin they attach to:

  • Audio Cornflower blue — stereo audio signal
  • MIDI Lime green — MIDI events (notes, CC, pitch bend, etc.)
  • Param Orange — block-rate control signal (for modulating knobs)
  • Signal Amber — audio-rate control signal (sample-accurate modulation)

You can only connect pins of matching kinds — an audio output won't plug into a MIDI input. Param and Signal are both for modulation, the difference being how fine-grained the modulation is. Most uses just need Param; Signal is for when you want sample-rate precision (e.g., audio-rate FM or audio-rate filter modulation).

See Signals & Pin Kinds for a deeper explanation of when to use each kind.

Making a cable

Click and drag from an output pin to an input pin. While you're dragging, valid drop targets (pins of the matching kind on other nodes) light up bright yellow. Release the mouse button over a yellow pin to make the connection. Release over empty space or an invalid pin to cancel.

Moving around

  • Pan: middle-click drag, or hold space and drag
  • Zoom: mouse wheel (zooms toward the cursor)
  • Fit all: the "Fit" button in the toolbar, or the Fit All hotkey
  • Select a node: click it
  • Box-select: click empty space and drag
  • Delete selected: Delete key

Adding nodes

Right-click an empty area of the graph for the node creation menu. The menu is organized by category (Instruments, Effects, Signal Shape, etc.). Some nodes open an editor immediately on creation (like the Waveform synth).

Node params

Most nodes have knobs and sliders that appear directly on the node as horizontal bars with labels. Drag them to change values, double-click to reset to default. If a param has an automation lane recorded, or is driven by an incoming Param/Signal cable, it appears locked and greyed out — the cable or automation is in control until you remove it.
