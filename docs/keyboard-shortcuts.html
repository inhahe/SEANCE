Keyboard Shortcuts

Home

File

  • Ctrl+N — New Project
  • Ctrl+O — Open Project…
  • Ctrl+S — Save Project

Edit

  • Ctrl+Z — Undo
  • Ctrl+Y or Ctrl+Shift+Z — Redo
  • Delete — Delete selected notes/nodes/links

Computer Keyboard MIDI

When the Keyboard MIDI toggle is on and the main window has focus:

  • A S D F G H J K L — white keys (starting at C)
  • W E T Y U O P — black keys
  • Z / X — octave down / up
  • Shift+key — loud (velocity 120)
  • Alt+key — soft (velocity 50)
  • Plain key — medium (velocity 90)

Graph navigation

  • Pan: middle-click drag
  • Zoom: mouse wheel (centered on cursor)
  • Fit all: the Fit button in the toolbar, or assigned hotkey
  • Box select: click and drag in empty space

Hotkeys

Most SoundShop2 actions can be bound to custom keyboard shortcuts or MIDI triggers via Options → Assign Hotkeys…. Dynamic per-node actions (mute/solo/open-editor on a specific track) can also be bound.
